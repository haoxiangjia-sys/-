# 主机配置与托运方案

## 配置目标
- **CPU**: AMD Ryzen 7 9800X3D
- **GPU**: NVIDIA RTX 5070 Ti
- **机箱**: 大机箱（ATX）
- **散热**: 水冷（AIO 360/420mm 或分体水）
- **用途**: 游戏 / 学习
- **地点**: 中国组装 → 暑假后飞机托运到英国

---

## 核心矛盾：大机箱 + 水冷 ≠ 飞机托运

大机箱（全塔/中塔）和一体水冷在飞机托运场景下有几个致命问题：

### 1. 航空公司液体限制
- **AIO 一体式水冷**：内部有冷却液（通常 200-400ml），部分航空公司**禁止托运含液体的电子产品**
  - 中国民航规定：单个容器超过 100ml 的液体**不能随身携带**，但托运需要看具体航司政策
  - 英国航空/中国国际航空等：内含水冷系统的电脑可能被拒绝托运，或在安检处被拦下
  - **分体水冷更麻烦**——漏液风险大，基本不适合托运

### 2. 玻璃侧板
- 大机箱大多有钢化玻璃侧板，托运几乎必碎
- 即使拆下单独包好，放在托运行李中也极易在搬运过程中碎裂

### 3. 体积 & 重量
- 全塔机箱托运通常超尺寸（长+宽+高 > 158cm，不同航司标准不同）
- 超重费用高昂，而且搬运过程粗暴，内部硬件可能损坏

## 推荐方案对比

### 方案 A：拆散运输 ✅（推荐）

将主机拆成核心部件，分散携带，到英国再重新组装。

| 部件                | 处理方式                    | 放哪里                   |
| ----------------- | ----------------------- | --------------------- |
| **CPU + 主板 + 内存** | 保持组装状态，包防静电袋            | **托运行李**（用衣物包裹）       |
| **显卡 (5070 Ti)**  | 拆下，放**原盒**或防静电袋+气泡膜     | **随身行李**（最贵最脆弱，随身最安全） |
| **M.2 SSD**       | 拆下或留在主板上都行              | 随主板托运                 |
| **电源**            | 单独包好                    | 托运行李                  |
| **机箱**            | **空箱托运**（内部塞满衣物缓冲）**或** | 托运 / 国内卖掉 / 到英国买新箱    |
| **水冷 (AIO)**      | **拆下**，单独包装托运           | 托运行李（注意液体限制）          |
| **风扇**            | 留在机箱上或拆下                | 随箱托运                  |

**到英国后需购买**：
- 新机箱（英国的 prices: £80-200 可以买到不错的 ATX 机箱）
- 可能需要新水冷（如果托运损坏或不让带）

### 方案 B：在英国买部分部件 🇬🇧

- 国内买：**CPU + 主板 + 内存 + SSD**（这些轻且容易携带）
- 到英国买：**机箱 + 电源 + 散热器**
- **显卡 (5070 Ti)**：国内买 OR 到英国买（英国定价含 VAT 20%，需看差价）

#### 为什么 CPU/主板/内存/SSD 值得国内买？
- 9800X3D 国内价格通常比英国好
- 这些部件体积小，随身带方便
- 不受托运风险影响

#### 为什么机箱/电源值得英国买？
- 机箱在英国买，省去托运的巨大麻烦和运费
- 电源（尤其 ATX 3.1 / 高瓦数）英国好买，不用背
- 英国电压 230V 与国内 220V 接近，电源支持宽幅电压（100-240V），**完全兼容**

### 方案 C：全部托运 + 买保险

- 如果实在想整机带过去：
  - 拆掉**显卡**和**水冷**，内部用**填充泡沫**塞满
  - **玻璃侧板拆下**随身带（或提前换成亚克力/金属侧板）
  - 购买**航空运输保险**（声明价值，保丢保损）
  - 到机场办**超规行李托运**（oversized baggage counter，不是普通托运）
  - **风险仍然很高**——不推荐

## 行李清单（方案 A，推荐）

### 🎒 随身行李（登机背包）
- ✅ RTX 5070 Ti（原盒或防静电袋+气泡膜，放背包最中间）
- ✅ 所有硬盘（如果还有 2.5" SATA SSD / HDD）
- ✅ 重要线材（如果背包有空位）

### 🧳 托运行李（28寸行李箱）
- ✅ 主板（带 CPU+内存+SSD）→ 防静电袋 → 衣物包裹，平放箱底
- ✅ 电源 → 原盒或气泡膜 → 放在衣物中间
- ✅ AIO 水冷冷排+风扇 → 气泡膜包裹
- ✅ 各种线材、配件盒

### 📦 可选的额外托运（纸箱）
- ✅ 空机箱（内部塞满衣物/泡沫）→ 超规行李托运
- ❌ 或：国内二手卖掉，英国买新机箱

## 电器兼容性注意事项
- **电源**：现代电源都是宽幅电压（100-240V），在英国直接用，只需换电源线（英标插头，几块钱）
- **显示器**：同上有宽幅电压的话，只需换电源线或加转接头
- **插头**：英国是 **G 型插头**（三脚方头），国内是 A/I 型，建议提前准备几根英标电源线或转换插头

## 结论：推荐方案 A + 部分 B 混合

1. **国内买** CPU 9800X3D + 主板 + 内存 + SSD + 显卡 5070 Ti
2. **随身带** 显卡 + 硬盘
3. **托运** 主板套装 + 电源 + 其他小件
4. **国内卖掉** 机箱 （或送人）
5. **到英国买** 新机箱 + 新水冷/风冷

这样既享受了国内的价格优势，又避免了托运大机箱和水冷的巨大风险。
